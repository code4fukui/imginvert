# imginvert

imginvertは、画像の反転、グレースケール化、コントラストと明るさの調整を行うWebアプリです。

## デモ
https://code4fukui.github.io/imginvert/

## 機能
- 画像の反転
- グレースケール化
- コントラストの調整
- 明るさの調整
- 調整した画像のダウンロード

## 使い方
1. ブラウザにファイルをドロップするか、画像をロードします。
2. スライダーを操作して画像の反転、グレースケール、コントラスト、明るさを調整できます。
3. 「save」ボタンをクリックすると、調整した画像をダウンロードできます。
