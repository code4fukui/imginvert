# imginvert

A simple image inversion tool.

## Demo
https://code4fukui.github.io/imginvert/

## Features
- Invert image colors
- Adjust grayscale, contrast, and brightness
- Save filtered image

## Usage
1. Drop an image file onto the page
2. Use the sliders to adjust the image filters
3. Click the "save" button to download the filtered image
